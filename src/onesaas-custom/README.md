# OneSaaS Custom

이 폴더는 **고객 커스텀 영역**입니다.

## 규칙

- 이 폴더의 파일은 **절대 자동 업데이트되지 않습니다**
- 랜딩 페이지, 커스텀 페이지 등을 이 폴더에 추가하세요
- 템플릿 업데이트 시에도 이 폴더는 안전합니다

## 구조 예시

```
onesaas-custom/
├── landing/          # 커스텀 랜딩 페이지
├── pages/            # 추가 페이지들
└── components/       # 커스텀 컴포넌트
```

## 시작하기

1. `landing/` 폴더에 커스텀 랜딩 페이지 컴포넌트를 추가하세요
2. `pages/` 폴더에 추가 페이지를 만드세요
3. 필요한 컴포넌트는 `components/`에 추가하세요
